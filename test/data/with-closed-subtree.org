Per interfacciarsi a Github via ssh, bisogna eseguire una semplice procedura.

Innanzitutto, verificare che ci sia un ssh-agent:

#+begin_src bash
$ eval "$(ssh-agent -s)"
> Agent pid 59566
#+end_src

Quindi, generare una SSH key:

#+begin_src bash
$ ssh-keygen -t ed25519 -C "example@maildomain.com"
#+end_src
